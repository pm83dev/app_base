Struttura del progetto

Il progetto è organizzato in due aree principali: backend e frontend.

  • backend: contiene un'app ASP.NET Core Web API che espone endpoint per gestire il contatore e fornire informazioni sullo stato del server.
  • frontend: contiene un'app Angular che consuma gli endpoint del backend, gestisce lo stato con NGXS e presenta l'interfaccia utente.

Il backend è configurato per consentire le richieste CORS dal frontend, facilitando lo sviluppo e il testing locale.
